# 合成

関数境界をドメイン概念に合わせる。小さくすることが目的ではない。

## いつ/いつでない

- 中間値にドメイン名がつく → 分離は自然
- つかない → 分割は人工的。迷ったら分割しない
- ドメインがわからないときはユーザーに確認する
