# Agent Runtime Config

This directory documents mutable AI tool operation.

Most runtime settings are intentionally not generated by Nix:

- `~/.codex/config.toml`
- `~/.claude/settings.json`
- `~/.apm/apm.yml`
- tool-installed skills, plugins, hooks, and MCP entries

Global instruction files are managed by Home Manager symlinks:

- `~/.codex/AGENTS.md` -> `dot-config/agents/shared-instructions.md`
- `~/.claude/CLAUDE.md` -> `dot-config/agents/shared-instructions.md`

Nix may still enable stable agent tool packages, such as Codex and Claude Code.
Tool behavior, shared prompts, and reviewed examples belong here.

Use this directory for shared instructions, reviewed examples, notes, and migration snippets.
The live source of truth is the tool-owned config file unless a specific
file here says it is managed by Home Manager or meant to be copied manually.

Promotion rule: keep disposable tools and settings outside Nix. Promote a tool
to nixpkgs or a maintained community package source only after it has been useful
for at least one month and rebuild-time management is worth the maintenance cost.
