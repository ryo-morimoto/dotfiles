# Agent Runtime Config

This directory documents mutable AI tool operation.

Runtime settings are intentionally not generated by Nix:

- `~/.codex/config.toml`
- `~/.claude/settings.json`
- `~/.apm/apm.yml`
- tool-installed skills, plugins, hooks, and MCP entries

Use this directory for reviewed examples, notes, and migration snippets only.
The live source of truth is the tool-owned config file unless a specific
file here says it is meant to be copied manually.

Promotion rule: keep disposable tools and settings outside Nix. Promote a tool
to `nix-config/packages/` only after it has been useful for at least one month
and rebuild-time management is worth the maintenance cost.
